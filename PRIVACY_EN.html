Privacy Policy

AI Comment Moderator for YouTube Studio
Last Updated: June 29, 2026 · Version 1.0

Overview

AI Comment Moderator is a Chrome extension that helps moderate YouTube Studio comments based on AI-powered criteria. This privacy policy explains what data is collected, how it is used, and your rights.

1. Data This Extension Collects

Comment Text

What: The text content of YouTube comments from pages you visit while using this extension.

Why: To evaluate comments according to your moderation criteria using external AI providers.

❌ Important: This extension does NOT store comment text and has no server of its own. The text is sent to the AI provider you choose (Groq or Pollinations) for the sole purpose of classifying it. Once the text leaves your browser, how that provider handles or retains it is governed by that provider's own privacy policy, which we do not control and cannot guarantee. Please review their policies (linked below) before using the extension.

Cached decisions: Only moderation decisions (not the original text) are saved locally on your device using chrome.storage.local (~10 MB max). The text is never copied or saved.

User Preferences & Settings

What: Your chosen AI provider, API keys (encrypted), moderation criteria, language preference, and other settings.

Why: To configure the extension according to your preferences.

Duration: Until you uninstall the extension or manually delete the data.

Storage: Locally on your device only. API keys are encrypted before storage.

Analytics Events

What: Anonymous event counts (e.g., "API key added") sent to an analytics service. The API key is NOT sent — only the count of how many times someone saves a key is tracked.

Why: To understand how the extension is used and improve it.

What is NOT included: User identifiers, IP addresses, browsing history, personal data, account information, or API keys.

Cookies: None. Analytics requests are sent without cookies — the analytics service does not set or store any cookies through this extension.

Learn more: See umami.is/privacy

❌ Data This Extension Does NOT Collect

  • Your YouTube account information or credentials
  • Your browsing history or IP address
  • Personal identification data
  • Cross-site tracking data

2. Third-Party Services

This extension sends comment text to the following AI providers for classification:

  • Groq — groq.com/privacy
  • Pollinations AI — pollinations.ai/privacy
  • Umami (for anonymous analytics) — umami.is/privacy

By using this extension, you accept that comment content will be sent to these services for processing. Each has its own privacy policy — review them if you have concerns.

3. Data Security

  • Encryption: API keys are encrypted with AES-256-GCM before storage.
  • Local storage: All settings and cached decisions remain on your device.
  • No own server: This extension does not operate its own server or store data in the cloud (except anonymous analytics events sent to the analytics service).

4. Your Rights

  • Local data: All data stays on your own device — no remote copy is held that you would need to request access to.
  • Delete: You can clear cached decisions at any time by clicking "🗑 Reset memory" in the Settings tab.
  • Uninstall: Uninstalling the extension from Chrome removes all stored data.
  • Opt-out of analytics: This extension currently does not provide a toggle for analytics. If you object, you may uninstall the extension.

5. Changes to This Policy

This policy may be updated as the extension evolves. Changes will be reflected in the version number above. Major changes will require your re-acceptance via the onboarding screen.
