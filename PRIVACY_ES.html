Política de Privacidad

AI Comment Moderator for YouTube Studio
Última actualización: 29 de junio de 2026 · Versión 1.0

Descripción General

AI Comment Moderator es una extensión de Chrome que ayuda a moderar comentarios de YouTube Studio basándose en criterios impulsados por IA. Esta política de privacidad explica qué datos se recopilan, cómo se usan y cuáles son tus derechos.

1. Datos que Recopila esta Extensión

Texto de Comentarios

Qué: El contenido de texto de comentarios de YouTube desde páginas que visitas mientras usas esta extensión.

Por qué: Para evaluar comentarios según tu criterio de moderación usando servicios de IA externos.

❌ Importante: Esta extensión NO almacena el texto de los comentarios y no tiene servidor propio. El texto se envía al proveedor de IA que elijas (Groq o Pollinations) con el único fin de clasificarlo. Una vez que el texto sale de tu navegador, lo que ese proveedor haga con él o cuánto tiempo lo conserve se rige por la propia política de privacidad de ese proveedor, que no controlamos ni podemos garantizar. Revisa sus políticas (enlazadas más abajo) antes de usar la extensión.

Decisiones en caché: Solo las decisiones de moderación (no el texto original) se guardan localmente en tu dispositivo usando chrome.storage.local (~10 MB máx). El texto nunca se copia ni se guarda.

Preferencias del Usuario y Ajustes

Qué: Proveedor de IA elegido, claves API (cifradas), criterios de moderación, preferencia de idioma y otros ajustes.

Por qué: Para configurar la extensión según tus preferencias.

Duración: Hasta que desinstales la extensión o borres manualmente los datos.

Almacenamiento: Solo en tu dispositivo. Las claves API se cifran antes del almacenamiento.

Eventos de Analítica

Qué: Conteos anónimos de eventos (ej. "clave API añadida") enviados a un servicio de analítica. La clave API NO se envía — solo se cuenta el número de veces que alguien guarda una clave.

Por qué: Para comprender cómo se usa la extensión y mejorarla.

Lo que NO incluye: Identificadores de usuario, direcciones IP, historial de navegación, datos personales, información de cuenta, ni claves API.

Cookies: Ninguna. Las peticiones de analítica se envían sin cookies — el servicio de analítica no establece ni almacena ninguna cookie a través de esta extensión.

Más información: Ver umami.is/privacy

❌ Datos que esta Extensión NO Recopila

  • Tu información de cuenta de YouTube o credenciales
  • Tu historial de navegación o dirección IP
  • Datos de identificación personal
  • Datos de rastreo entre sitios

2. Servicios de Terceros

Esta extensión envía texto de comentarios a los siguientes proveedores de IA para clasificación:

  • Groq — groq.com/privacy
  • Pollinations AI — pollinations.ai/privacy
  • Umami (para analítica anónima) — umami.is/privacy

Al usar esta extensión, aceptas que el contenido de los comentarios se enviará a estos servicios. Cada uno tiene su propia política de privacidad — revísalas si tienes dudas.

3. Seguridad de los Datos

  • Cifrado: Las claves API se cifran con AES-256-GCM antes del almacenamiento.
  • Almacenamiento local: Todos los ajustes y decisiones en caché permanecen en tu dispositivo.
  • Sin servidor propio: Esta extensión no opera su propio servidor ni almacena datos en la nube (excepto los eventos anónimos enviados al servicio de analítica).

4. Derechos del Usuario

  • Datos en local: Todos los datos permanecen en tu propio dispositivo — no existe ninguna copia remota a la que tendrías que solicitar acceso.
  • Eliminar: Puedes borrar decisiones en caché en cualquier momento haciendo clic en "🗑 Borrar memoria" en la pestaña Ajustes.
  • Desinstalar: Desinstalar la extensión de Chrome elimina todos los datos almacenados.
  • Opt-out de analítica: Actualmente esta extensión no proporciona una opción de desactivación para analítica. Si te opones, puedes desinstalar la extensión.

5. Cambios en Esta Política

Esta política puede actualizarse a medida que la extensión evoluciona. Los cambios se reflejarán en el número de versión anterior. Los cambios importantes requerirán tu re-aceptación a través de la pantalla de incorporación.
